====
TTim
====
:Author: Mark Bakker
:Website: http://code.google.com/p/ttim/

TTim is a multi-layer transient analytic element solver for modeling
groundwater flow.

Dependancies
------------

TTim depends on:

 - Python 2.x
 - numpy
 - scipy
 - matplotlib

Build
-----

To build TTim, a Fortran compiler is requird.
